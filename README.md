ВАРИАНТ № 3

    В процессе функционирования АСУ  ВД  в  файле  фиксируются данные о самолетах, совершивших посадку. Каждая запись имеет структуру типа:

          ТУ-154М      Б-3726       131           12.1
          марка ЛА    бортовой     количество    груз
                       номер        пассажиров

 1) подготовить программу, сортирующую записи с использованием индексной сортировки линейным методом в порядке возрастания числовой части бортовых номеров; определить суммарное количество пассажиров, вес грузов; результаты печатать в виде таблицы;
 2) обеспечить входной контроль бортового номера, количества пассажиров и веса груза, выполнить отладку и тестирование.
Чтение данных их файла производить с использованием функций ввода/вывода языка С++.
Алгоритм должен быть параметризован; обмен данными с подпрограммой должен осуществляться только через параметры;  исходные  данные  хранятся в отдельном файле.
